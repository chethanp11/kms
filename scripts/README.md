# Scripts

Build, validation, and maintenance scripts live here.

Available helpers:

- `dev-api.sh` - run the API against the repo-local paths
- `dev-worker.sh` - run the worker against the repo-local paths
- `dev-kmi.sh` - start the KMI frontend
- `dev-infopedia.sh` - start the Infopedia frontend
- `dev-stack.sh` - start API and worker together
- `health-check.sh` - verify the API health endpoints
